# User Manual

## 1. Admin Guide

### Super Admin

- Sign in with a super admin account
- Review dashboard metrics
- Create and manage users under `Users`
- Review role permissions under `Roles & Permissions`
- Monitor audit trails under `Audit Logs`
- Review system master data and configuration under `Settings`

### SBA Admin

- Register, update, and approve MSMEs and BDSPs
- Generate reports and exports
- Review data quality and sync backlogs
- Oversee opportunities and product approvals

## 2. Data Entry Guide

- Go to `MSME Registry` or `BDSP Registry`
- Select `New MSME` or `New BDSP`
- Complete all required fields
- Save the record in draft or submit through workflow
- Use `Offline Data Collection` when field connectivity is unavailable
- Sync drafts once the device reconnects

## 3. County Supervisor Guide

- Open `County Dashboard`
- Filter to your county
- Review pending verification and correction items
- Open individual MSME or BDSP records
- Use the verification/workflow screens to:
  - return for correction
  - verify
  - approve
  - reject

## 4. Analyst Guide

- Use `Executive Analytics Dashboard` for national trends
- Use `County Dashboard` for local analysis
- Use `Reports` to generate monthly, quarterly, annual, and custom outputs
- Use `Exports` through the report interface or the export API for downstream analysis

## 5. Report Generation Guide

- Open `Reports`
- Select report title, type, period, and desired format
- Generate the report
- Review historical entries and download stored exports

## 6. GIS and Map Guide

- Open `Map & GIS`
- Switch between MSME and BDSP layers
- Click map points to view location-level detail
- Use this page for verification planning and geographic overview

## 7. Notifications Guide

- Open `Notifications`
- Review assignments, approvals, report events, and sync issues
- Mark items read individually or in bulk
